# Privacy Policy

The Khuym plugin is a repo-local skill bundle. It does not operate a hosted service and does not collect telemetry on behalf of `hoangnb24`.

When you install or use the plugin locally, any prompts, repository content, command output, or generated artifacts are handled by the AI tooling and local integrations you choose to run. Data handling for those systems is governed by their own privacy policies and your local environment.

This repository stores only the plugin code, skill content, and documentation required to run Khuym workflows. If you fork or modify the plugin, you are responsible for the privacy posture of your own distribution.

Questions about this repository can be directed through the project issue tracker at <https://github.com/hoangnb24/skills/issues>.
